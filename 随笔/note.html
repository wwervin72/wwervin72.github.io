# import

import 加括号("url")引入css和js会采用异步的方式加载，如果采用这种方式可能就需要注意下 css 的优先级或者是 js 执行的上下文环境

# keyframe 动画

编写 keyframe动画对时候最好是写完整的 1%到100%的动画样式，或者是 from, to的形式，如果只写一部分，在ie下可能会导致意想不到的后果

举个例子：一个dialog弹出框

.dialog {
  position: absolute;
  top: 20%;
  left: 50%;
  transform: translate(-50%, -50%);
  animation: zoomIn;
}
// 下面这种方式会导致两个ie下有两个连续的动画 第一个是scale的动画，另一个是 translate的
@keyframes zoomIn {
  0%	{
    opacity: 0;
    transform: scale(1);
  }
  1%	{
    opacity: 1;
    transform: scale(.01);
  }
}
// 推荐方式 
@keyframes zoomIn {
  0%	{
    opacity: 0;
    transform: translate(0, 0) scale(1);
  }
  100%	{
    opacity: 1;
    transform: transalte(-50%, 0) scale(.01);
  }
}

# ie9 大文件引入

webpack 打包的开发环境通常是将css js 等打入到一个文件，通常是比较大的，在 ie9 下有文件大小的限制，不会读取炒锅4000kb的文件，所以可以使用 css-split-webpack-plugin 插件去进行文件的拆分

const CSSSplitWebpackPlugin = require('css-split-webpack-plugin').default;

module.exports = {
    plugins:[
    	new CSSSplitWebpackPlugin({
            size: 3800, //默认值4000
            filename: 'css/[name]-[part].[ext]'
        }),
    ],
};

# div 等元素无法触发 keyup 等事件

主要原因是因为默认情况下这些元素无法聚焦，我们可以通过给这些元素设置 tabindex 让目标元素可以聚焦，以及控制它是否/在何处参与顺序键盘导航（通常使用Tab键，因此得名）其中 tabindex 有三个可选值：负值、0、正值

  1. 负值：通常设置成 -1，表示元素可以聚焦，但是不能通过键盘导航来访问该元素
  2. 0：表示元素可以聚焦，并且可以通过键盘导航来访问该元素，访问顺序与 dom 结构顺序一致
  3. 正值：表示元素可以聚焦，也可以通过键盘导航来访问该元素，但是这种设置通过键盘导航访问的顺序通常是很滞后的

如果在 div 上设置了 tabindex，那么它的子元素内容不能通过箭头键来滚动，除非也加上 tabindex 属性

# 前端处理302临时重定向

Ajax 请求无法处理302的状态码，需要同步的http 请求浏览器才能够识别到 response header 里面的 location 字段去进行页面跳转

在一些鉴权的场景下，后端可能返回302 重定向到登陆界面或者是返回 401(这个axaj 就自己能处理，进行前端跳转登陆)。

有一些场景下登陆界面需要一些后端才能拿到的数据，这时候就需要通过 http 请求访问后端，由后端去动态构造重定向地址，前端浏览器在拿到 http 请求后会自动跳转

# 资源加载错误执行操作

通常标签在加载资源失败(比如未找到资源)后，会去执行onerror事件，比如link, script, img 等这时候我们可以在onerror事件中去执行一些操作

举个例子：

图片加载失败后，展示默认的图片

<img src="asdb.png" onerror="javascript:this.src='https://t7.baidu.com/it/u=1595072465,3644073269&fm=193&f=GIF'">

Link 样式脚本加载失败后，执行js代码

<link rel="stylesheet" href="abc.css" onerror="javascript:console.log(this);">

需要注意的是：如果是 script 标签，则不能加载他错误事件里预设的资源。由于性能等因素，这个是不被标准采纳的。
