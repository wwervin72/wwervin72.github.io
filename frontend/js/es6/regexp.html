# RegExp构造函数

# 1. 构造函数

    // ES5中如果第一个参数是正则表达式, 就不允许使用第二个参数添加修饰符
    new RegExp(/abc/ig, 'i')  //报错
    // ES6第二个参数的修饰符会覆盖第一个参数正则表达式的修饰符
    new RegExp(/abc/ig, 'i').flags
