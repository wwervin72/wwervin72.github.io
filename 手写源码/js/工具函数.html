# 实现一些工具函数

# 1. once

该函数具有以下特点：

1. 接收一个 function 类型作为参数，并且返回一个 function
2. 返回的 function 只执行一次，多次执行返回的结果和第一次执行一样

利用闭包来实现，下面是实现源代码

function once(func) {
    let invoked, result
    if (typeof func !== 'function') {
        throw new TypeError('请传入一个方法')
    }
    return function (...args) {
        if (!invoked) {
            invoked = true
            result = func.apply(this, args)
        }
        return result
    }
}

# 2. setTimeout

该函数和 window.setTimeout 是一样的，具有以下特点：

1. 可以接受无限多个参数，他的参数分别是这样
    1.1 第一个参数是一个方法
    1.2 第二个参数是一个时间毫秒值，setTimeout 会在延迟这个值的时间后执行传入的方法
    1.3 后面的系列参数将会被传入到第一个参数的方法里执行
2. 返回一个 timer id，使用它可以清除还未执行的定时器

这里使用 window.requestAnimationFrame 来实现该方法

function mySetTimeout(func, delay, ...args) {
    let timer, now = Date.now()
    function loop() {
        if (Date.now() - now >= delay) {
            func.apply(this, args)
            return
        }
        timer = window.requestAnimationFrame(loop)
    }
    timer = window.requestAnimationFrame(loop)
}
