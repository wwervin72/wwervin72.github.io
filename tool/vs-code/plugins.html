# vscode plugins

1. material theme：编辑器主题
2. material icon theme：图标主题
3. beautify：格式化代码
4. Prettier
5. auto close tag：自动添加闭合标签
6. auto rename tag：自动修改闭合标签名
7. Path Intellisense: 路径提示
8. Bracket Pair Colorizer ：让代码的各种括号呈现不同的颜色。
9. code runner：在编辑器里直接运行代码 查看结果
10. Color Picker ：可以直接在编辑器里打开色板，选择各种模式的颜色
11. Document This ：可以给函数、类等自动的加上详细的注释
12. Git History ：方便的查看git版本管理的详细信息
13. Live Server ：可以一键在本地启动服务器
