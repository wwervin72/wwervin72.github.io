# Intersection Observer

# 什么是 Intersection Observer

他提供了一种异步观察目标元素与其祖先元素或顶级文档视口（viewport）交叉状态的方法。其祖先元素或视口被称为根。

当一个 IntersectionObserver 对象被创建时，其被配置为监听根中一段给定比例的可见区域。一旦 IntersectionObserver 被创建，则无法更改其配置，所以一个给定的观察者对象只能用来监听可见区域的特定变化值；然而，你可以在同一个观察者对象中配置监听多个目标元素。

以上定义来自MDNhttps://developer.mozilla.org/zh-CN/docs/Web/API/IntersectionObserveropen in new window，大概意思就是我们能够使用它来监听一片可见区域，并且观察该区域内的某一个对象，如果该对象相对于我们监听的可见区域位置发生了变化，则会触发他的回调函数。

# 用法

先来一个例子：

<!DOCTYPE html>
<html lang="en">
<head>
 <meta charset="UTF-8">
 <meta http-equiv="X-UA-Compatible" content="IE=edge">
 <meta name="viewport" content="width=device-width, initial-scale=1.0">
 <title>Document</title>
 <style>
  .flag {
   background-color: #f00;
   color: white;
   font-size: 20px;
   font-weight: bold;
  }
  .item {
   height: 1000px;
  }
 </style>
</head>
<body>
 <div class="list">
  <div class="item">1</div>
 </div>
 <div class="flag">资源加载中...</div>
 <script>
  var list = document.querySelector('.list')
  var count = 1;
  var observer = new IntersectionObserver(([entry]) => {
   if (entry.intersectionRatio <= 0) return;
   var item = document.createElement('div')
   item.classList.add('item')
   item.innerText = ++count;
   console.log('懒加载');
   list.appendChild(item)
  }, {
   rootMargin: '0px',
   threshold: [0.1],
  })
  observer.observe(document.querySelector('.flag'))
 </script>
</body>
</html>

构造函数 IntersectionObserver()，提供两个参数，一个是触发事件后的回调函数，另一个是可选的配置

callback:

entries: 是一个包含观察数据的对象数组，里面包含的下面的一些信息

    1. boundingClientRect：观察区域的边界 rect 信息
    2. time：可见性发生变化的时间，是一个高精度时间戳，单位为毫秒
    3. target：被观察的目标元素，是一个 DOM 节点对象
    4. intersectionRect：观察对象与监听区域的交叉区域 rect 信息
    5. intersectionRatio：观察对象处于监听区域的可见比例
    6. intersectionRect 在 boundingClientRect 里面的大小比例，当完全在里面时为1，完全不在时为0

options[可选]:

1. root:监听区域，默认为根元素
2. rootMargin: 默认值为"0px 0px 0px 0px"，使用css定义方法，基于top, right, bottom, left四个方向定义值，大于 0 时，放大监听区域大小，即监听区域本身的大小再加上这个值，作为边界。小于0时，缩小边界
3. threshold: 是一组0-1的数字数组，他表示观察元素在监听区域的大小占比，达到这些值时会触发回调函数

找了几张图方便理解这几个参数：

分享一个很好的例子理解option的这两个属性 https://codepen.io/michellebarker/pen/xxwLpRGopen in new window

返回值：

返回一个 IntersectionObserver 实例。我们调用它的 observe 开始观察指定目标。他有下面四个方法：

    1. disconnect：IntersectionObserver对象停止监听区域
    2. observe：指定观察某一个目标
    3. takeRecords：返回所有观察目标的对象数组
    4. unobserve：停止监听指定目标
