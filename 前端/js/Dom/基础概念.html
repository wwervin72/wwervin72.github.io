# 1. children、childNodes、parentNode、parentElement

DOM Tree 中包括三种类型的节点：元素节点、文本节点（换行，空格等都算）、属性节点

# 1.1 children

返回当前节点下的所有子元素节点

# 1.2 childNodes

返回当前元素的所有类型的子节点，也包括 换行、文本、注释等，不包括属性节点

# 1.3 parentNode 父节点 parentNode 父元素

举个例子

document.body.parentNode // html
document.body.parentElement // html

document.body.parentNode.parentNode // document
document.body.parentElement.parentElement // null
