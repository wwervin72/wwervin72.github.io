# 7. includes() startsWith() endsWith()

includes()：返回布尔值，表示是否找到了参数字符串。 startsWith()：返回布尔值，表示参数字符串是否在源字符串的头部。 endsWith()：返回布尔值，表示参数字符串是否在源字符串的尾部。

    var s = 'Hello world!';

    s.startsWith('Hello') // true
    s.endsWith('!') // true
    s.includes('o') // true

# 8. repeat

返回一个新的字符串，表示将原字符串重复n次

    'x'.repeat(3) // "xxx"
    'hello'.repeat(2) // "hellohello"
    'na'.repeat(0) // ""

# 9. padStart padEnd

es2017引入了字符串补全长度的功能，如果字符串的长度不够指定的长度，会在头部或者尾部补全，padStart用于头部补全，endStart用于尾部补全

    'x'.padStart(5, 'ab') // 'ababx'
    'x'.padStart(4, 'ab') // 'abax'
    
    'x'.padEnd(5, 'ab') // 'xabab'
    'x'.padEnd(4, 'ab') // 'xaba'
    
    //如果原字符串的长度大于或者等于指定的最小长度，则返回原字符串
    'xxx'.padStart(2, 'ab'); // 'xxx'
