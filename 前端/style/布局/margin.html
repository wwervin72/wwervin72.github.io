  • 不带定位

marginopen in new window 是常见的一种 css 样式布局属性，我们用它来设置指定元素的外边距，他是四个属性（ margin-topopen in new window，margin-rightopen in new window，margin-bottomopen in new window，和 margin-leftopen in new window）的简写形式。

它有三个可选值：

  1. length 固定的数值大小
  2. percentage 百分比大小
  3. auto 让浏览器自己选择一个合适的外边距，在特定情况下可以使用这种方式来让指定元素居中

今天这里我们主要来说下如果赋值给 margin 百分比 会是怎么样的情况，也即是讨论下百分比的实际大小计算参照对象

先来举几个例子

# 不带定位

<div class="root">
  <div class="parent">
    <div class="child"></div>
  </div>
</div>

.root {
  position: relative;
  width: 500px;
  height: 500px;
  margin-left: 200px;
  background: linear-gradient(to right, red 50%, blue 50%);
}
.parent {
  width: 300px;
  height: 300px;
}
.child {
  width: 100px;
  height: 100px;
  background-color: #ff0;
  margin-left: 50%;
}

这里我们可以看出来，不带定位 child 的 margin-left 的值为父元素 (.parent元素) 的 width * percentage

# 带定位

<div class="root">
  <div class="parent">
    <div class="child"></div>
  </div>
</div>

.root {
  position: relative;
  width: 500px;
  height: 500px;
  margin-left: 200px;
  background: linear-gradient(to right, red 50%, blue 50%);
}
.parent {
  width: 300px;
  height: 300px;
}
.child {
  position: absolute;
  width: 100px;
  height: 100px;
  background-color: #ff0;
  margin-left: 50%;
}

这里我们可以看出，child 的 margin-left 实际值为定位参照（.root元素）对象的 width * percentage

# 结论

margin 给百分比值时的计算方式为：

  1. 目标元素的定位属性值为 static 或者 relative 时，百分比计算的参照对象为父节点
  2. 目标元素的定位属性值为 absolute 或者 fixed 时，百分比计算的参照对象为定位的参照对象
